# Odin Recipes

This project is a basic recipe website built as part of the [The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations) curriculum. The main objective was to demonstrate proficiency in basic HTML structure and Git workflow.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/recipes-top/)

## 🛠️ Technologies Used
*   **HTML5:** Semantic structure, lists, anchor tags, and images.
*   **Git:** Version control and commit history.
*   **GitHub:** Repository hosting.

## 📝 Features
*   A main landing page linked to multiple recipe pages.
*   Usage of ordered and unordered lists to display ingredients and steps.
*   Proper directory structure for images and pages.

## 🧠 What I Learned
During this project, I reinforced my knowledge of:
*   Setting up the HTML boilerplate.
*   Working with relative paths for links and images.
*   Using semantic HTML tags for better accessibility.
*   Committing code with meaningful messages.
